# 🧠 Difference in Easy Words

## 🔧 1. BIOS (Basic Input Output System)

👉 **What it is:**
A **settings screen for hardware**

👉 **When it works:**
Before Windows starts

👉 **What you do here:**

* Check if hardware is detected (battery, HDD, RAM)
* Change boot order
* Enable/disable devices

👉 **Simple meaning:**
🟢 *“BIOS is where you check and control hardware before OS loads”*

---

## 🧪 2. F10 Diagnostics (Hardware Test)

👉 **What it is:**
A **testing tool for hardware**

👉 **When it works:**
Before Windows (like BIOS)

👉 **What you do here:**

* Test RAM
* Test HDD/SSD
* Test battery
* Check for hardware failure

👉 **Simple meaning:**
🟢 *“Diagnostics is used to test if hardware is working or faulty”*

---

## ⚙️ 3. Boot Menu

👉 **What it is:**
A **one-time selection screen**

👉 **When it works:**
During startup

👉 **What you do here:**

* Choose where to boot from:

  * HDD/SSD
  * USB
  * Network

👉 **Simple meaning:**
🟢 *“Boot menu is used to choose how to start the system”*

---

## 🛡️ 4. Safe Mode (Windows feature)

👉 **What it is:**
A **limited version of Windows**

👉 **When it works:**
After OS loads (inside Windows)

👉 **What you do here:**

* Troubleshoot driver issues
* Remove problematic software
* Check if issue is software-related

👉 **Simple meaning:**
🟢 *“Safe Mode runs Windows with minimum drivers to find software problems”*

---

| Feature   | BIOS             | Diagnostics   | Boot Menu          | Safe Mode            |
| --------- | ---------------- | ------------- | ------------------ | -------------------- |
| Type      | Settings         | Testing tool  | Selection screen   | Troubleshooting mode |
| Runs when | Before OS        | Before OS     | During boot        | Inside OS            |
| Purpose   | Control hardware | Test hardware | Choose boot device | Fix software issues  |

# 🧠 Real engineer shortcut

* 🔍 **Check hardware detected?** → BIOS
* 🧪 **Check hardware working?** → Diagnostics
* 🔄 **Change boot device?** → Boot Menu
* 🛠️ **Fix software issue?** → Safe Mode

# ⚡ 🔑 SHORTCUT KEYS + PROCESS

## 🔧 1. BIOS (Setup)

### 🔑 Shortcut:

* **F1 / F2 / Fn + F2** (common in Lenovo)

### 📍 Process:

1. Restart laptop
2. Immediately keep pressing **F2 (or Fn+F2)**
3. BIOS screen will open

---

## 🧪 2. Diagnostics (F10 / Lenovo Diagnostics)

### 🔑 Shortcut:

* **F10** *(some models)*
* OR via Boot Menu

### 📍 Process:

1. Restart system
2. Press **F10** continuously
3. Select **Diagnostics**

👉 Alternate:

* Enter Boot Menu → choose Diagnostics

---

## ⚙️ 3. Boot Menu

### 🔑 Shortcut:

* **F12** (most common in Lenovo)

### 📍 Process:

1. Restart system
2. Press **F12** continuously
3. Select:

   * HDD/SSD
   * USB
   * Network

---

## 🛡️ 4. Safe Mode (Windows)

### 🔑 Shortcut (working method):

* **Shift + Restart**

### 📍 Process:

1. Go to Start Menu
2. Hold **Shift** → Click **Restart**
3. Select:

   * Troubleshoot
   * Advanced Options
   * Startup Settings
   * Restart
4. Press:

   * **4 → Safe Mode**
   * **5 → Safe Mode with Network**

---

## ⚡ BONUS (Forced method if system not booting)

1. Turn ON system
2. Force shutdown (hold power button)
3. Repeat **3 times**

👉 Windows will enter recovery → then go to Safe Mode
